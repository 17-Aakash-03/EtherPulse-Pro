## 🔍 PROJECT OVERVIEW
**EtherPulse** is a high-fidelity security dashboard designed to de-anonymize blockchain behavior and provide real-time threat intelligence. In a decentralized environment where anonymity often hides malicious activity, EtherPulse provides a **"Behavioral Intelligence Layer"** that translates raw transaction data into actionable forensic signatures.

---

## 🛡️ CORE LOGIC & HEURISTICS
The system utilizes a proprietary **Heuristic Analysis Engine** to monitor the "pulse" of the Ethereum network, focusing on three critical security vectors:

### 1. Forensic Radar [Visualizer]
A multi-vector neural graph that generates a behavioral "fingerprint" of any Ethereum address. It evaluates **Security Risk**, **Whale Volume**, **Bot Activity**, and **DNA Integrity**.

### 2. Sentinel HUD [Frontend]
A maximalist, industrial-tech terminal interface built for high data density. It features a real-time **Pulse Monitor** that tracks the Ethereum network’s heartbeat and gas congestion baselines.

### 3. Whale Intelligence [Mobile Link]
Integrated via the **Twilio API**, this module pushes encrypted forensic alerts directly to a user's WhatsApp when significant liquidity shifts ($1M+ movement) are detected.
